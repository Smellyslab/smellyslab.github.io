About


A coding language is a way to create code using commands and variables in strings. Lucylang is a very simple coding language to help kids learn and work their way up to real languages like Python or html without using block code. This coding language is greatly simplified, and is made using Python so it will help people learn Python by using the same commands and terminal. Variables are things you can assign other meanings, for example a = 2. Commands are things you can tell the computer to do, like wait 1 second or color text.


About the creators. Connor Jones, he is the founder and creator for lucy. He came up with the idea when he got bored one day during a coding session and wanted to create a new coding language. He thought, "Why not make one with Python3?" Nate Bergh is a massive part of the whole thing. He's the main creator of this website and the big html writer. He is a great friend of Connors and has been there to help create and innovate with lucy since the start.
