Lucy Lang Functions / Syntax Docs

We made this coding language to show that there is something better than block code for kids to learn on. As kids, when we were first introduced to block code, we felt dissapointed because we expected to learn real line code. We created this languge, Lucylang, so kids could learn real line code without it being overcomplicated and hard to learn. We hope you enjoy and learn from our efforts


Variables: addvariable.varname.varvalue, takes values after each. and stores them in json file as vars, variable.print.value.varname, pulls variable stored under specified name where it says varname and prints its value.


Sleep: sleep.time, sleeps for the amount of time (in seconds) specified where it says time must be an int


Execution: cmd.run.command, runs the command as if user was running it in command must be something like echo, syntax: cmd.run.echo hello // would echo hello to the terminal


Printing: sys.out.none:stuff to print, prints everything defined after the :, sys.out.red: prints everything after : in red text color., sys.out.green: like red and none prints text after : in green color


Terminal Functions: clear, just clears the terminal...
